BaseTest

global abstract class BaseTest
Description BaseTest is an abstract base class that defines a single test in Cinnamon.

When a Cinnamon test runs, Cinnamon framework manages the entire lifecycle of a test execution and executes the following methods (setup, test and teardown) during the test execution. Context context must be used to store and retrieve test fixture used during test execution.
