Wait

global with sharing class Wait
Description Wait is a class that allows tests to wait until some ExpectedCondition turns true. See the sample code below on how to use this class:


cinnamon.ExpectedCondition condition =
    cinnamon.ExpectedConditions.textToBePresentInElement(locator, 'h1.title');
cinnamon.Wait wait = accountPageObject.getWait();
wait.until(condition);
Tests can get a Wait object by invoking PageObject.getWait() method.
