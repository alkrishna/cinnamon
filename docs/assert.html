Assert

global class Assert
Description A set of assertion methods useful for writing Cinnamon tests It is desirable to use Cinnamon's assert methods to assert a condition in your Cinnamon test, rather than using the standard Apex's System.assert methods.

Methods

  assert

global static void assert ( String message, Boolean condition)
Description Asserts that the specified condition is true. If it is not, an error is returned that causes Cinnamon test execution to halt. The tearDown method in your Cinnamon test still gets executed even when its execution stops due to assertion failure.
Parameters
message - custom message displayed as part of the error message
condition - the condition to be evaluated
