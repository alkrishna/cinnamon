String apply()
Description The string returned is a javascript snippet to be executed on selenium server. The javascript should test for the expected condition and be formatted as an anonymous, self-executing function, explicitly returning a boolean. For example: to test if an element is inthe DOM: (function(){var el = selenium.browserbot.findElementOrNull('#id'); return (el != null);}())
Returns String - a javascript snippet to be executed on selenium server.
