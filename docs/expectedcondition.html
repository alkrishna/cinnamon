ExpectedCondition

global interface ExpectedCondition
Description ExpectedCondition models a condition that is expected to eventually evaluate to something that turns true or false.
Examples include determining if a web page has loaded or that a certain element becomes visible or invisible.

Methods

apply

String apply()
Description This method returns a javascript snippet that gets sent and executed on a remote selenium server. The javascript should test for the expected condition and be formatted as an anonymous, self-executing function, explicitly returning a boolean.

For example: to test if an element is in the DOM:


(function(){
    var el = selenium.browserbot.findElementOrNull('#id');
    return (el != null);
}())
Returns a string value that represents a javascript snippet to be executed on selenium server.
